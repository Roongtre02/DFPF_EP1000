Autodesk Fusion 360 | Assignment 1: design phrase

This week, I went through Assignment 1 designs and start modelling it out using fusion 360

Assignment 1 - Design

I wanted to design a box with sliding door and 2 pull out drawer with separation panel on the middle

Creation - Model making

1. Create New Sketch - I drew different squircle for inner body shell and sliding track on the model base

2. Extrude surface - Create different height for sit in inner model partition and sliding track

3. Mirror model base to the top

4. Create new sketch and sketch out middle panel

5. Extrude & move to construction plane

6. Create Sketch under sheet metal tool & extrude the inner model surface

7. Turn on the middle panel layer back & combine both shell and middle panel. [Target: shell, Tool bodies: middle panel; cut]

8. Create new parameters

9. Create drawer base & extrude

10. Project the drawer base to draw out the side panel

11. Mirror side panel

12. Copy bottom drawer to the construction plane

13. Unfold door to flat surface

14. Create new sketch on door surface & project on the surface. Turn off the body layers & sketch out the lines with dimensions shown below

15. Create new parameters

16. Select rectangular pattern as shown below, repeat the same thing for the container inner shell

17. Change model space to manufacture and create mnufacturing model

18. Create sketch plane

19. Arrange models on the plane

20. Export file under fabrication post process and save to DXF file.

Final model (one side of sliding door open for better inner view)
