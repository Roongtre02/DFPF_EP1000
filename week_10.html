Embeded Programming

Embeded: programming of small computers to operate device / drive

Programming: code / language to communicate with device

Principle of programming:

Sequencing: Order in which instructions occurs and are being processed

Selection: Determines which path a program takes when it is running

Iteration: Repeated execution of a section of the code when the program is running

Signals / data:

Digital: Discrete or binary representation of data etc: on/off states, logic levels, or numerical values using binary encoding, possible values: 0 or 1

Tasks: reading button states, controlling digital sensors, or interacting with digital devices or interfaces

Analog: Continuous or variable representation of data etc: temperature, voltage, sound intensity, or physical measurements, over range or continuous scale

Tasks: analog sensors, interfacing with analog devices, or performing tasks that require continuous or variable data

Exercise 13 - Blink an external LED

1. Create TinkerCad profile and create new circuit board

2. Add Arduino Uno R3

3. Link LED positive side to 8, and add a resistor to the negative side. Connect resistor to GND.

4. Change resistance to 220 ohm
