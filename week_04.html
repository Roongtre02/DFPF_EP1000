AUTODESK FUSION 360 | Rectangle pattern, Shell

This week, we are learning Autodesk Fusion 360. From the lesson, I learnt how to create multiple similar pattern and shell an object using rectangle pattern.

Exercise 5

1. Create a rectangle & extrude surface

2. Create new sketch & draw a circle on the extruded surface, then extrude surface

3. Select rectangular pattern & key in the overall distance and quantity to different axis

4. Select bottom surface and shell inside by 1.2mm

5. Create new surface on bottom top surface & draw 2 circle

6. Select the surface between 2 circle, right click and choose press pull, then left click on the bottom surface(highlighted)

7. Rectangular pattern

8. Fillet the extruded part by 0.2mm

Fusion view

Exercise 6

1. Create a rectangle and fillet all 4 corner by 15mm

2. Draw 2 circle on 1 side

3. Extrude the inner circle surface & choose cut

4. Shell the stadium shape surface by 1.5mm

5. Extrude the outer circle surface (join)

5. Create new sketch and add text on the surface

6. Press/pull text by 1mm

Fusion view

Exercise 7

1. Create a 20*20*20mm cube

2. Select rectangular pattern & key in the overall distance and quantity to different axis

3. Extrude each cube to designated height

4. Combine all bodies & create a new components

5. Shell bodies by 4mm

Fusion view

Exercise 8

1. Create a cylinder & shell it by 2.5mm

2. Create handle by drawing 3 rectangle & extrude by 6mm thk

3. Fillet outter corner by 100mm, inner corner by 5mm

4. Select move tool and place it on the handle, rotate 90 degrees

5. Move handle attaching the cup from top view

6. Arrange height from front view
