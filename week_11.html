Embedded Programming 2 , Design Thinking

This week, we are learning how to design or craft solutions to solve particular problems we met in daily lifes or at certain times. Besides, we also did some activities to come out with some ideas with draft sketches and explain it to the lecturer.

Design thinking

SPS Framework

1. Syntactic Analysis

Understanding of how things are made in terms of materials, the dimensional scale.

2. Pragmatic Analysis

Understanding of how things work and operates taking into consideration of the ergonomics factor

3. Semantic Analysis

Understanding of how the product interacts with the context, its form analogies inspiration and its spiritual connesction to the user.

Assignment 3 - Tinkercad

Task 1

1. Connect a LED with a current limiting resistor (220 ohm) to port 6 of the Arduino Uno Board.

2. Write a program to blink the LED in a variety of pattern, i.e. Blink 1 time, delay(Milliseconds) then Blink 2 time etc.

Task 2

1. Using the same technique, connect 3 LEDs of different colours to the Arduino board using pins 6,7, & 8.

2. write a program to produce running lights i.e the LEDs are lit in sequence e.g. R -> G -> B -> R -> G -> B and so on.

Task 3

1. Add a switch to the circuit in task 1. (you should use a pull-up resistor of 10K if you are operating in INPUT mode.)

2. Write a code when the switch is pressed, the LEDs will ight up. Pressing the switch again will turn the LEDs off.

Task 4

1. Use the switch in task 3 to control the type of light-up and flashing of the LEDs, for example:

- Start with LEDs are all in the OFF state.

- Pressing the switch, turns on only the RED LED.

- Pressing the switch again, turns on only the GREEN LED.

- Pressing the switch again, turns on only the BLUE LED.

- Pressing the switch again, turns on all LEDs

- Pressing the switch again, turns off all LEDs returning to first state.
