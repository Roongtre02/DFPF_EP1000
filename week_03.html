GIMP, INKSCAPE, AUTODESK FUSION 360

This week, we are learning new software (GIMP, Inkscape and Autodesk Fusion 360). From the lesson, I learnt that GIMP is a cross-platform image editor for task such as photo retouching, image composition and image authoring. As for Inkscape, it allow users to create logos with an image reference. Lastly, Autodesk Fusion 360 software allow users to built 3D devices from scratch.

GIMP

1. Download GIMP

2. Upload chosen image into GIMP for edit

3. Select "Free select tool" and select the outer line of the object in the image

4. right click & invert selection

5. right click edit layer & choose add alpha channel

6. Delete the selection

7. Export file to png

INKSCAPE

1. Download INKSCAPE

2. Upload downloaded image into inkscape

3. Press (B) to select Draw bezier curves and straight line tool and trace the image

4. Press (B) & change shape to Ellipse

5. Add feathers and beak outline

6. Turn off import image layer and export to png

Autodesk Fusion 360

Exercise 3

1. Create new sketch on x plane, draw a rectangle accoring to dimension shown

2. Draw another rectangle and use dimension tool to create exact measurements shown

3. Create 2 circle from the middle point of the rectangle

4. Trim the excess / unwanted lines & done

Exercise 4

1. Create new sketch on x plane, draw two rectangle accoring to dimension shown

2. Draw two circle using 2-tangent circle

3. Draw polygon and outer circle on both sides

4. Trim the excess / unwanted lines & done
